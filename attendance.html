📋 출석부 사용 설명서
관리자용 · 비밀번호: RCC2024
① 기본 사용 흐름
1
날짜 선택 — 상단 날짜 입력칸을 누르면 달력이 표시됩니다. 오늘 날짜는 자동으로 입력되어 있습니다.
2
시간 선택 — 시간 입력칸에 모임 시작 시간을 입력하거나 드롭다운에서 선택합니다. (예: 09:00)
3
행사 시작 버튼 클릭 — 해당 날짜·시간으로 출석 세션이 열립니다.
4
이름 터치/클릭 — 명단에서 이름을 누르면 출석 ↔ 미출석이 토글됩니다. 보라색 = 출석, 회색 = 미출석.
5
출석 마감 — 행사 시작 후 5시간이 지나면 자동으로 수정이 잠깁니다.
💡 상단 통계 칩(전체/출석/미출석)을 누르면 해당 목록만 필터링됩니다.
② 명단 관리 (명단 탭)
1
카톡 대화 붙여넣기 — 카카오톡 대화방에서 대화 내용을 복사한 뒤 텍스트 박스에 붙여넣고 이름 추출 버튼을 누르면 자동으로 이름만 골라냅니다.
2
직접 추가 — 하단 이름 입력칸에 이름을 쓰고 추가 버튼을 누릅니다.
3
이름 클릭 (통계 보기) — 명단 탭에서 이름을 클릭하면 해당 회원의 출석일수 · 출석률 · 노쇼 횟수를 팝업으로 확인할 수 있습니다.
4
검색 — 상단 🔍 검색창에 이름 또는 초성으로 빠르게 찾을 수 있습니다.
③ QR코드 출석
1
QR 탭으로 이동 — 상단 탭에서 QR을 선택합니다.
2
QR 출석 토글 ON — 스위치를 켜면 QR 이미지가 생성됩니다.
3
QR 화면 출력 — 태블릿·노트북 화면에 QR을 띄우거나 프린트해서 입구에 비치합니다.
4
참석자 스캔 — 참석자가 스마트폰 카메라로 QR을 스캔 → 이름 입력 → 출석 완료. 명단에 있는 이름만 인식됩니다.
⚠️ QR은 앱 주소 기반으로 하나만 생성됩니다. 행사마다 새로 만들 필요 없이 계속 재사용합니다.
④ 과거 기록 조회 · 삭제
1
과거 탭으로 이동 — 날짜별 출석 기록이 최신순으로 표시됩니다.
2
카드 클릭 — 해당 행사의 출석자/미출석자 명단을 펼쳐볼 수 있습니다.
3
기록 삭제 — 카드 우측의 삭제 버튼을 누르면 해당 날짜 기록이 삭제됩니다. 삭제 이력은 변경 탭에서 확인 가능합니다.
4
통계 탭 — 전체 출석 랭킹 상위 20명과 월별 평균 출석률을 확인합니다.
⑤ 구글 시트 연동 (선택 사항)
💡 출석 데이터를 구글 시트에 자동 저장하려면 아래 순서대로 한 번만 설정하면 됩니다.
[ 1단계 ] 구글 시트 만들기
1
브라우저에서 sheets.google.com 접속 후 구글 계정으로 로그인합니다.
2
왼쪽 상단 ＋ 비어있는 스프레드시트를 클릭해 새 파일을 만듭니다.
3
파일 이름을 RCC 출석부 등 원하는 이름으로 변경합니다 (선택).
[ 2단계 ] Apps Script 코드 등록
4
상단 메뉴에서 확장 프로그램 → Apps Script를 클릭합니다. 새 탭이 열립니다.
5
왼쪽 파일 목록에서 Code.gs를 클릭하고, 편집창의 기존 내용을 모두 지웁니다.
6
아래 코드 파일(Code.gs)의 전체 내용을 복사해서 붙여넣은 뒤 💾 저장(Ctrl+S)합니다.
※ Code.gs 파일은 출석부 앱과 같은 폴더에 있습니다.
메모장으로 열어서 전체 선택(Ctrl+A) → 복사(Ctrl+C)하세요.
[ 3단계 ] 시트 초기화
7
Apps Script 편집창 상단 함수 선택 드롭다운에서 initSheets를 선택합니다.
8
▶ 실행 버튼을 클릭합니다. 처음에는 권한 허용 팝업이 뜨면 허용을 누릅니다.
9
구글 시트로 돌아가면 Members · Attendance · Settings 세 개의 탭이 자동 생성되어 있습니다.
[ 4단계 ] 웹 앱으로 배포
10
Apps Script 편집창 오른쪽 상단 배포 → 새 배포를 클릭합니다.
11
유형 선택 아이콘(⚙️)을 클릭 → 웹 앱을 선택합니다.
12
설명란에 원하는 내용 입력(예: 출석부 v1), 아래 설정을 확인합니다:
· 다음 사용자로 실행: 나(내 계정)
· 액세스 권한: 모든 사용자
13
배포 버튼 클릭 → 권한 허용 팝업이 뜨면 계정 선택 후 허용을 누릅니다.
14
배포 완료 화면에 표시되는 웹 앱 URL을 복사합니다. (https://script.google.com/... 형태)
[ 5단계 ] 출석부 앱에 URL 입력
15
출석부 앱으로 돌아와서 시트 탭으로 이동합니다.
16
구글 시트 연동 토글을 ON으로 켭니다.
17
복사한 URL을 Apps Script URL 입력칸에 붙여넣고 저장을 누릅니다.
18
🔄 전체 데이터 구글 시트에 동기화 버튼을 눌러 기존 데이터를 한 번에 올립니다.
✅ 이후부터는 출석 체크 시 자동으로 구글 시트에 저장됩니다.
⚠️ URL이 바뀌면(재배포 시) 앱에도 새 URL을 다시 입력해야 합니다. 배포는 되도록 처음 한 번만 하세요.
